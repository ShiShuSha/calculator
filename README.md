# FastAPI Calculator

дз нетология

## Описание

Методы:
- сложение
- вычитание
- умножение
- деление
- вычисление сложных выражений

## Установка

Install dependencies:


pip install -r requirements.txt


## Запустить приложение

uvicorn main:app --reload


Открыть в браузере: http://127.0.0.1:8000/docs

## Как загрузить файлы (если используешь терминал)

git init

git add .

git commit -m "FastAPI calculator homework"

git branch -M main

git remote add origin https://github.com/USERNAME/fastapi-calculator.git

git push -u origin main
