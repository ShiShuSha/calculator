# FastAPI Calculator

дз нетология

## Описание

Методы:
- сложение
- вычитание
- умножение
- деление
- вычисление сложных выражений

## Установка

Install dependencies:


pip install -r requirements.txt


## Запустить приложение

uvicorn main:app --reload


Открыть в браузере: http://127.0.0.1:8000/docs
